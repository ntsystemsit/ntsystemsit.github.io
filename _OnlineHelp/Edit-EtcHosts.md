# Edit-EtcHosts

## SYNOPSIS
Edit /etc/hosts file with notepad.

## SYNTAX

```
Edit-EtcHosts
```

## DESCRIPTION
This funtion starts notepad.exe as administrator and opens the hosts file for editing.
If this function is running on PowerShell Core, it runs "sudo vi /etc/hosts"
